Aplicativo de sistema de facturación. Taller SENA

¿EN QUE SE BASA EL APLICATIVO?

Este aplicativo se desarrolló buscando solucionar los errores del aplicativo que fue entregado por el profesor, acá se solucionó tanto errores técnicos, como errores de ortografía y la generación de la factura.


¿COMO FUNCIONA?

En este caso el archivo es ejecutable mediante el editor de archivos ejecutando el html en un navegador o mediante un editor de codigo como lo es VS(Visual Studio Code) siempre y cuando se tenga la extensión de live server.
Al entrar en el aplicativo se visualizará un recuadro con un formulario que contiene casillas explicando que digitar. Hay algo corregido acá y es que es obligatorio poner un id o un numero, ya que al insertar letras, el software no dejará digitarlo, lo mismo sucede con la cantidad de días minimos, ya que al insertar un número menor a "2", el aplicativo lanzará un error y no lo dejará continuar hasta que cambie ese valor.

¿POR QUÉ SE USÓ I.A EN CIERTOS PROCESOS?

Se usó I.A solo para ajustar cambios y asegurar que el programa quede bien estructurado evitando errores en la generación de la factura.
